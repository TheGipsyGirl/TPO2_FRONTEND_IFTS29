## Proyecto

**Tafí Viejo English Academy**

Este proyecto consiste en desarrollar una Landing Page utilizando exactamente el mismo prompt en dos agentes de inteligencia artificial diferentes, con el objetivo de comparar cómo interpreta cada uno las instrucciones y analizar los resultados obtenidos.

## Análisis

Para la realización de este trabajo se utilizó exactamente el mismo prompt en ambos agentes. Los dos lograron generar una Landing Page funcional y cumplieron con los requisitos solicitados en la consigna, aunque cada uno resolvió el proyecto con un enfoque diferente.

### Claude Sonnet 4

**Aspectos destacados**

* Generó un diseño moderno y visualmente atractivo.
* Organizó muy bien la estructura del proyecto.
* Utilizó HTML semántico y un CSS bastante completo.
* Incorporó animaciones suaves y una buena distribución de los elementos.
* Incluyó varias buenas prácticas relacionadas con accesibilidad y SEO.

**Aspectos a mejorar**

* Algunas imágenes utilizadas no estaban disponibles y fue necesario reemplazarlas.
* Algunas rutas de archivos requirieron pequeños ajustes.
* En resoluciones pequeñas aparecieron algunos detalles de alineación que hubo que corregir.

### Codex GPT-5

**Aspectos destacados**

* Generó una Landing Page funcional y bien organizada.
* El código resultó claro y fácil de comprender.
* Interpretó correctamente el prompt.
* La estructura del proyecto quedó ordenada y sencilla de mantener.
* Presentó un buen comportamiento responsive.

**Aspectos a mejorar**

* El diseño fue más simple en comparación con Claude.
* Incorporó menos animaciones y efectos visuales.
* La interfaz resultó más tradicional.

---

## Comparación general

Aunque ambos agentes recibieron exactamente el mismo prompt, los resultados no fueron idénticos. Claude puso más énfasis en el diseño visual y en la experiencia de usuario, incorporando una interfaz más moderna y una mayor cantidad de detalles estéticos.

Por su parte, Codex se enfocó principalmente en cumplir los requisitos funcionales, generando una solución más simple, ordenada y fácil de entender.

Esto demuestra que un mismo prompt puede producir resultados diferentes según el agente utilizado y la forma en que cada modelo interpreta las instrucciones.

---

## Conclusión

Esta práctica me permitió comprobar la importancia que tiene el Prompt Engineering al momento de trabajar con herramientas de inteligencia artificial. A pesar de utilizar exactamente el mismo prompt, ambos agentes desarrollaron soluciones distintas, cada una con sus propias fortalezas.

En mi experiencia, Claude obtuvo un resultado más atractivo desde el punto de vista visual y con mayor nivel de detalle en el diseño. Codex, en cambio, generó una solución más simple y directa, con un código fácil de interpretar y mantener.

Considero que ambos cumplieron correctamente con la consigna y que esta experiencia permitió observar cómo la calidad del prompt influye en el resultado final, aunque cada agente aporte su propio enfoque al momento de desarrollar una solución.
